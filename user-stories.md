# User Stories

## User Story 1
As a user, I want to create a new habit so that I can track my daily activities.

## User Story 2
As a user, I want to view a list of my habits so that I can see what I am tracking.

## User Story 3
As a user, I want to mark a habit as completed so that I can monitor my progress.

## User Story 4
As a user, I want to edit an existing habit so that I can update incorrect information.

## User Story 5
As a user, I want to delete a habit so that I can remove habits I no longer follow.
